## / ei /   / ai /   / ɔi /   / au /   / əu /   / iə /   / eə /   / uə /   / ju /

<u>a</u>ge 年龄	f<u>a</u>ce 脸	l<u>a</u>ke 湖	c<u>a</u>ke 蛋糕
